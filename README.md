# Todo-App-Node.js
## How to Use:
###### * Clone the Repo and Run npm install.
###### * (Add Todo) Run - node app.js add --title="todo title here" --body="Note body here"
###### * (Remove Todo) Run - node app.js remove --title="todo title here"
###### * (Show Todo List) Run - node app.js list
###### * (Show Todo By Title) Run - node app.js read --title="todo title here"
